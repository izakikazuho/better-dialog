## ちょっといいモーダルを、ネイティブな`<dialog>`要素で作ろう

**Better Dialog** は、HTML ネイティブの `<dialog>` 要素を活かしつつ、モダンで柔軟なモーダル UI を構築できる軽量な JavaScript ライブラリです。

「ゼロから作り直す」のではなく、「ブラウザが持っている機能をより良く使う」ことを目指して設計されています。HTML セマンティクス、キーボード操作、アクセシビリティ対応など、標準に乗っかった実装を前提としながら、必要に応じてアニメーションやプラグインで拡張できます。

デザインシステム、業務アプリ、あるいはちょっとしたモーダルまで、Better Dialog は余計な依存を持たず、堅実なベースを提供します。

### 主な特徴

- ⚡ 軽量・依存なし
- 🎬 プラグインによる挙動やアニメーションの追加
- 🧩 独自ロジックによる拡張が可能
- 🌐 ネイティブ準拠、モダンブラウザ対応
- 🎨 スタイルなし —— お好きなデザインでどうぞ

## Better Dialog が追加する機能

Better Dialog は、ネイティブの `<dialog>` 要素に以下のような機能を加えています：

- バックドロップ（ダイアログ外）クリックによるクローズ処理
- アニメーションの注入と、完了後に実行されるライフサイクルフック
- プラグインによる拡張ロジックの注入（例：スクロールロック）
- `data-better-dialog-show` によるトリガー要素の自動バインド
