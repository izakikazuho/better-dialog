---
title: DialogItem
description: Represents a single dialog instance.
---

Each `<dialog>` element initialized by BetterDialog is wrapped in a DialogItem.

## Properties

- `el: HTMLDialogElement`
- `id: string`
- `isShow: boolean`
- `triggerEls: HTMLElement[]`
- `closeButtonEl: HTMLElement | null`

## Methods

### show()

Displays the dialog.

### close()

Closes the dialog.
